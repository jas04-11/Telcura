# Telcura- Healthcare Telecommunication App 🏥📞

A full-stack telemedicine platform that enables real-time communication between patients and doctors through chat and video calls. The application provides secure authentication, appointment scheduling, and a responsive user-friendly interface for seamless healthcare access.

---

## 🚀 Features

- 🔐 JWT-based User Authentication  
- 💬 Real-time Chat Communication  
- 📹 Video/Audio Calling Support  
- 📅 Appointment Scheduling System  
- 👨‍⚕️ Doctor & Patient Dashboard  
- 📱 Responsive UI Design  
- ⚡ Fast and Scalable Backend APIs  

---

## 🛠️ Tech Stack

**Frontend:** React.js, HTML5, CSS3, Bootstrap  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**Real-Time Communication:** Socket.io, WebRTC  
**Authentication:** JWT, bcrypt  
**Version Control:** Git, GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

git clone https://github.com/your-username/Telcura.git

### 2️⃣ Navigate to Project Folder

cd Telcura

---

## 🔧 Backend Setup

cd backend
npm install
npm start

Create `.env` file inside backend folder:

PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key

---

## 💻 Frontend Setup

cd mediconnect_vite
npm install
npm run dev
